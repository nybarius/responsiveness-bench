# Responsiveness Bench v0.1.0

The initial release provides an executable operational core for scoring whether a response addresses the explicit and implicated proposition layers of a claim.

## Included

- A deterministic, valence-blind scoring kernel.
- Typed claim and response primitives.
- Left/right/neutral structural-family validation.
- Accuracy and invariance metrics.
- Deterministic JSONL input and output.
- A command-line interface.
- An 18-case seed corpus covering six response structures.
- Annotation, contribution, and methodological documentation.

## Deliberate limits

This release does not infer annotations from raw prose, adjudicate disputed facts, call a language model, or claim population-level benchmark results. The next empirical milestone is a naturalistic corpus with independent annotation and inter-annotator-agreement measurement.
